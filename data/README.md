# Data

Place the HPA training images in this folder.

Download from Kaggle:
https://www.kaggle.com/c/human-protein-atlas-image-classification/data

Expected file naming per sample:
- `{Id}_red.png`
- `{Id}_green.png`
- `{Id}_blue.png`
- `{Id}_yellow.png`

Total: ~31,072 samples × 4 channels = ~124,288 PNG files (~22 GB)
